# Installation

In an existing Docusaurus project, install the plugin:

<Tabs>
  <TabItem value="npm" label="npm" default>
    ```bash
    npm install --save @r74tech/docusaurus-plugin-panzoom
    ```
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    ```bash
    yarn add @r74tech/docusaurus-plugin-panzoom
    ```
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    ```bash
    pnpm add @r74tech/docusaurus-plugin-panzoom
    ```
  </TabItem>
  <TabItem value="bun" label="Bun">
    ```bash
    bun add @r74tech/docusaurus-plugin-panzoom
    ```
  </TabItem>
</Tabs>

<br />

Add the plugin to your Docusaurus configuration file:

```js title="docusaurus.config.js"
module.exports = {
  plugins: [
    '@r74tech/docusaurus-plugin-panzoom'
  ],
};
```
